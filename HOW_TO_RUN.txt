  YOUTH ALCHEMY — AI Skincare Platform
  Setup & Run Guide
═══════════════════════════════════════════════════

STEP 1 — Install Python dependencies
─────────────────────────────────────
  pip install -r requirements.txt

STEP 2 — (Optional) Set up Ollama for AI plan generation
──────────────────────────────────────────────────────────
  Install Ollama from: https://ollama.com
  Then run:
    ollama serve
    ollama pull llama3.2

STEP 3 — Start the server
──────────────────────────
  python run.py

  The app will be available at: http://localhost:5000

─────────────────────────────────────────────────────

FEATURES
════════

🔬 FACE SCAN (AI Skin Analysis)
  • Click "Get Started" on the home page
  • Sign up / Log in
  • Open camera → Capture → Analyse Skin
  • Fill in your skin profile → Generate Plan
  • Results shown in Quick Summary + Detailed tabbed view

📅 APPOINTMENT BOOKING (Fully Functional)
  • Visit the "Book an appointment" section on the home page
  • Enter name, email, phone, date, and pick a live time slot
  • Slots update in real-time (booked slots disappear)
  • Double booking is prevented at the database level
  • Success / error messages shown inline

🔐 ADMIN PANEL
  • Go to: http://localhost:5000/#admin
  • Enter admin token: youthalchemy_admin_2024
  • View all bookings, update status, export CSV
  • Change token via env var: ADMIN_TOKEN=your_token

─────────────────────────────────────────────────────

API ENDPOINTS
═════════════
  POST /api/appointments               — Book an appointment
  GET  /api/appointments/availability  — Get available time slots for a date
  GET  /api/appointments               — Admin: view all bookings (requires X-Admin-Token header)
  PATCH /api/appointments/:id/status   — Admin: update booking status

  POST /api/scan                       — CV face analysis (requires auth)
  POST /api/generate                   — AI skin plan (requires auth + Ollama)
  POST /api/signup                     — Create account
  POST /api/login                      — Log in

─────────────────────────────────────────────────────

ENVIRONMENT VARIABLES (optional)
══════════════════════════════════
  PORT=5000              Server port (default 5000)
  ADMIN_TOKEN=secret     Admin panel password
  YOUTHALCHEMY_DB_PATH   Custom SQLite path
